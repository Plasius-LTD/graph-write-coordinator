# @plasius/graph-write-coordinator

Queue-first write coordinator with hot-key batching and operation tracking.

## Installation

npm install @plasius/graph-write-coordinator

## Development

npm install
npm run build
npm test

## Architecture

- Cross-package architecture is tracked in plasius-ltd-site/docs/adrs/adr-0020 through adr-0024.
